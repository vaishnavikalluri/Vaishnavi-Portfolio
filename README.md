# 🌟 Vaishnavi Kalluri - Premium Portfolio

A modern, elegant, and minimal personal portfolio website built with **Next.js 15+ (App Router)**, **Tailwind CSS v4**, and **Framer Motion**.

## 🎨 Design Philosophy
- **Theme**: Soft neutral beige (`#f5f3ef`) with muted green accents (`#2f6f64`).
- **Typography**: Playfair Display for headings (elegant) and Inter for body text (clean).
- **Feel**: Premium, airy, with smooth curves and subtle micro-animations.

## 🚀 Features
- **Fully Responsive**: Mobile-first design.
- **Smooth Scrolling**: Enhanced with CSS and Framer Motion.
- **SEO Optimized**: Pre-configured metadata and open graph tags.
- **Modern Components**: Sticky glassmorphism navbar, animated project cards, and clean forms.
- **Production Ready**: Structured for easy deployment.

## 📁 Project Structure
```text
/app         - Main pages and layout
/components  - Reusable UI elements (Button, Navbar, etc.)
/sections    - Individual page sections (Hero, About, etc.)
/public      - Static assets and images
/styles      - Global styling
```

## 🛠️ Getting Started

### 1. Installation
Ensure you have Node.js installed, then run:
```bash
npm install
```

### 2. Development
Start the local development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view it.

### 3. Build
Create a production optimized build:
```bash
npm run build
```

## 🌍 Deployment

### Deploy to Vercel (Recommended)
1. Push your code to a GitHub repository.
2. Go to [Vercel](https://vercel.com) and click **"New Project"**.
3. Import your repository.
4. Vercel will automatically detect Next.js settings. Click **"Deploy"**.

### Deploy to Netlify
1. Push your code to GitHub.
2. Go to [Netlify](https://netlify.com) and select **"Add new site"** -> **"Import from Git"**.
3. Connect your GitHub and select the repo.
4. Set Build command to `npm run build` and Publish directory to `.next`.
5. Click **"Deploy site"**.
