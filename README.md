# Vaishnavi Kalluri - Portfolio

A modern, elegant, and minimal personal portfolio website built with **Next.js 15+ (App Router)**, **Tailwind CSS v4**, and **Framer Motion**.

##  Design Philosophy
- **Theme**: Soft neutral beige (`#f5f3ef`) with muted green accents (`#2f6f64`).
- **Typography**: Playfair Display for headings (elegant) and Inter for body text (clean).
- **Feel**: Premium, airy, with smooth curves and subtle micro-animations.

##  Features
- **Fully Responsive**: Mobile-first design.
- **Smooth Scrolling**: Enhanced with CSS and Framer Motion.
- **SEO Optimized**: Pre-configured metadata and open graph tags.
- **Modern Components**: Sticky glassmorphism navbar, animated project cards, and clean forms.
- **Production Ready**: Structured for easy deployment.
